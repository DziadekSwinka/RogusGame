# RogusGame
RoguśGame - gra o rogusiu 

## O grze

Oparta na faktach gra akcji przedstawia niezwykłą historię 
Rogusia, który musi walczyć z otaczającym go 
światem. Toczy walkę m.in. z chochołami i grzybami

## Sterowanie 

- ASWD - ruch postaci 
- Enter - strzał 
- E - podniesienie przedmiotu
- Tab - wytwarzanie
